# DX Language

Experimental language writen in crystal.

The follow code

```
# a :: Int
a = 5

print a

# b :: Int
b = a + 3

# print b

# c :: Int
c = a + b - 3

# print c

# d :: Int
d = a + c + 3 + -5

print d

e = "Hello World"

print e
```

Will print:

```
5
13
Hello World
```
